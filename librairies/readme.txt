Ici sont placés, les "librairies" utilisées dans mon programme arduino.
La bibliothèque SPI n'est pas comprise, vu qu'elle est installée de base sur le logiciel Arduino IDE.
Si ce n'est pas le cas il suffit d'aller dans: Croquis --> Inclure une bibliothèque --> Gérer les bibliothèques, puis il suffit de taper dans la barre de recherche SPI et d'installer les paquets, de tout fermer et de réouvrir arduino.
